# Open Stake - Crypto Casino Clone

An open-source, high-performance web platform featuring pixel-perfect clones of popular crypto casino games (Stake Originals). Built from the ground up with a focus on UI/UX, animations, and a fully functional provably fair system.

Since there are very few accurate, high-quality open-source implementations of these games available, this project serves as a comprehensive reference for building modern betting interfaces and implementing verifiable randomness.

## Features

- **Pixel-Perfect UI/UX**: Meticulously crafted interfaces matching the "Cyber Casino" aesthetic, complete with 3D glassmorphism elements, dynamic micro-animations, and responsive layouts.
- **Provably Fair Algorithms**: True deterministic game outcomes using cryptographic hashing (HMAC-SHA256, server seeds, client seeds, and nonces) identical to industry standards.
- **Fairness Verification**: Integrated debug modes and on-the-fly seed rotation/verification tools for every game, allowing players to verify the integrity of their bets.
- **Interactive Data Visualization**: Real-time play history, profit tracking, and deep statistical charts.
- **Modern Tech Stack**: Built with React, Vite, and Ant Design.

## Included Games

1. **Crash**: Real-time multiplier curve game with a dynamic charting system, auto-cashout mechanics, and live bet tracking.
2. **Plinko**: Physics-based board with customizable risk levels (Low, Medium, High) and varying row counts. Features an advanced dropping animation engine and live profit floating effects.
3. **Mines**: A deterministic, provably fair mine-sweeping game with customizable grid configurations, dynamic total profit calculators, and a fairness debug map.
4. **Dino**: An interactive, animated running game clone featuring advanced difficulty parameters (survive percentages) and an immersive day/night cycle transition based on the multiplier.

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tanh1c/stake-originals-clone.git
cd stake-originals-clone
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Technical Details

### Provably Fair Implementation
Every game utilizes a `ProvablyFair` utility class that handles cryptographic generation. Outcomes are generated by hashing a combination of:
- Server Seed (Unrevealed until rotation)
- Client Seed (User customizable)
- Nonce (Incremented per game)

This guarantees that the game cannot be manipulated by the house after the bet has been placed.

### State Management
The project utilizes advanced React Context and Hook patterns to handle real-time balance updates, bet validations, and global state across different game components seamlessly.
